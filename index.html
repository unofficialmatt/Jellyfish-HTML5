This starter repository allows you to spin up a new project based on the Jellyfish UI scss/js front-end framework.

Getting started

  1. Install dependencies with $ npm install.
  2. Remove the existing git repo with $ rm -rf .git.
  3. Initialise git with $ git init.
  4. Start watching for changes, initialise BrowserSync and more with $ gulp or $ gulp init. This will provision a local php server allowing you to run both HTML and php files.
  5. Use npm to install additional dependencies. Use gulp to add these to your workflow as required.

Jellyfish's default variables can all be overridden from the SASS partials in /src/scss/02-settings.

For best practice, your SASS partials should conform to the ITCSS methodology.
